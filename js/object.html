对象都有一个原型__proto__
获取对象属性的时候，如果没有找到，会原型链上找，一直找到父对象的原型链为止
每个对象的属性都是，[string] Key : [object] value

各种创建对象的区别：
直接使用 对象() 返回的是 { return 这个对象; }
使用 new 对象() 返回的是 { 除了 return 之外的内容; }
所有的定义顺序， function 参数 --> function 定义 --> var 定义；
如果先有参数 x ，再有 var x，则后面的 x 覆盖参数 x；
如果现有函数 x() ，再有定义 var x; 则 x 最终还是 function x()；如果之后 y 有做赋值等操作，则为操作后的对象
