js 数组是弱类型的，数组里面可以是任意对象和基本数据类型；
定义数组：
var arr = [***,***]
var arr = new Array()
var arr = new Array(10) 创建长度为 10 的数组，元素都是 undefined
var arr = new Array(1,2,3,4) 创建还有 1,2,3,4 四个数字的数组
使用 delete arr[index] 可以把对应 index 赋值为 undefined，长度不变，对应key值不存在
使用 arr[index] = undefined 可以把对应 index 赋值为 undefined，长度不变，对应key值依然存在
使用 arr[index] 可以为数组添加一个元素
使用 arr.push() 也可以为数组添加一个元素
arr.shift() 删除数组第一个元素
arr.unshift(val) 添加 val 作为 arr 的第一个元素
arr.length -= 1 删除数组最后一个元素
arr.pop() 删除数组最后一个元素
